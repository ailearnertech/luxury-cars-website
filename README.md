## 🚀 How to Run the Project

### Method 1: Run Locally

1. Download or Clone the repository:

```bash
git clone https://github.com/your-username/luxury-cars-website.git
```

2. Open the project folder.
3. Double-click `index.html`  
   OR  
   Right-click → Open with → Browser.

The website will run in your default web browser.

---

### Method 2: Run Using VS Code (Recommended)

1. Open the project folder in VS Code.
2. Install the **Live Server** extension.
3. Right-click on `index.html`.
4. Click **Open with Live Server**.

The project will run on a local development server.
